# Story 6.6: README Status Badges

## Status

Ready for Review

## Story

**As a** potential user or contributor evaluating WriteScore,
**I want** to see at-a-glance status indicators in the README,
**so that** I can quickly assess project health, security posture, and maintenance activity.

## Acceptance Criteria

### Badge Organization

1. Badges are organized into logical rows (status, quality, info)
2. Badges use consistent style (flat or flat-square)
3. All badges link to relevant detail pages

### CI/Build Status Badges

4. GitHub Actions CI badge shows current build status (existing, verify working)
5. CodeQL security scanning badge added showing analysis status
6. Pre-commit.ci badge added (if using pre-commit.ci service) OR pre-commit badge from shields.io

### Code Quality Badges

7. Codecov coverage badge displays current test coverage percentage
8. Code style badge indicates Ruff is used for linting/formatting
9. Type checking badge indicates mypy is used

### Security Badges

10. Security policy badge links to SECURITY.md
11. Dependabot status badge shows dependency update status
12. OpenSSF Scorecard badge (optional, if project qualifies)

### Project Info Badges (Lead Position)

13. Python version badge placed first (helps users quickly assess compatibility)
14. License badge placed second (existing, verify correct)
15. PyPI/Downloads badges omitted until package is published (avoid placeholder confusion)

### Maintenance Badges

16. Last commit badge shows recent activity
17. GitHub issues badge shows open issue count
18. PRs welcome badge encourages contributions

### Badge Count and Ordering

19. Total badge count limited to 8-10 most important (avoid visual overload)
20. Badge order: Python → License → CI → Coverage → Security → Maintenance

### Shields.io Style Options

All shields.io badges can use a style parameter:
- `?style=flat` (default)
- `?style=flat-square`
- `?style=plastic`
- `?style=for-the-badge`
- `?style=social`

Recommendation: Use default `flat` style for consistency with GitHub native badges.
